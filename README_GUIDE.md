# দারুল আমাল একাডেমি (Darul Amal Academy) ম্যানেজমেন্ট গাইড

আপনার ওয়েবসাইটের আধুনিকায়ন এবং অ্যাডমিন কন্ট্রোল সম্পন্ন হয়েছে। নিচে এর ব্যবহারের নিয়ম দেওয়া হলো:

## ১. অ্যাডমিন প্যানেল প্রবেশ (Login)
- **লিঙ্ক**: আপনার ফোল্ডারে থাকা `login.html` ফাইলটি ওপেন করুন।
- **পাসকোড**: বর্তমান পাসকোড দেওয়া আছে: `123456`
- **পরিবর্তন**: আপনি যদি পাসকোড পরিবর্তন করতে চান, তবে `login.html` ফাইলটি একটি কোড এডিটর (যেমন Notepad) দিয়ে ওপেন করে নিচের লাইনে গিয়ে নতুন সংখ্যা লিখে দিন:
  ```javascript
  if (pass === '123456') { // এখানে আপনার পছন্দের পাসওয়ার্ড লিখে দিন
  ```

## ২. কিউআর কোড (QR Code) তৈরি
- **লিঙ্ক**: আপনার ফোল্ডারে থাকা `create-qr.html` ফাইলটি ওপেন করুন।
- **ব্যবহার**: আপনার ওয়েবসাইটের ফাইনাল লিঙ্ক (যেমন: `darulamal.com`) বক্সে লিখে "কোড আপডেট করুন" বাটনে ক্লিক করুন।
- **ডাউনলোড**: কিউআর কোডটি তৈরি হলে তার উপর মাউস দিয়ে **Right Click** করে **"Save Image As..."** ক্লিক করে আপনার কম্পিউটার বা ফোনে সেভ করে নিন। এটি আপনি আপনার ব্যানার বা কার্ডে ব্যবহার করতে পারবেন।

## ৩. ফ্রিতে গুগলে যুক্ত করা (Hosting)
গুগলে ফ্রিতে আপনার সাইট দেখা যাবে যদি আপনি এটি নিচের যেকোনো একটি সার্ভারে আপলোড করেন (সবগুলো ১-২ মিনিটে কাজ করবে এবং একদম ফ্রি):

- **টিপস**: [Netlify Drop](https://app.netlify.com/drop) ওয়েবসাইটে গিয়ে আপনার `darul amal` ফোল্ডারটি মাউস দিয়ে টেনে সেখানে ছেড়ে দিন। সাথে সাথেই আপনার সাইট ইন্টারনেটে লাইভ হয়ে যাবে এবং একটি লিঙ্ক দেবে। এরপর গুগল অটোমেটিক এটি খুঁজে পাবে।

## ৪. ডাটাবেস (Database) সেটআপ এবং নিয়ন্ত্রণ
আমি আপনার সাইটে **Firebase Firestore** যুক্ত করার সব কোড লিখে দিয়েছি। এখন এটি আপনার নিজের নিয়ন্ত্রণে নিতে নিচের ধাপগুলো অনুসরণ করুন:

১. **Firebase একাউন্ট**: [Firebase Console](https://console.firebase.google.com/) এ গিয়ে আপনার জিমেইল দিয়ে লগিন করুন।
২. **প্রজেক্ট তৈরি**: 'Add Project' এ ক্লিক করে একটি প্রজেক্ট খুলুন (নাম দিন: Darul Amal)।
৩. **ওয়েব অ্যাপ**: 'Web' (</>) আইকনে ক্লিক করে একটি অ্যাপ রেজিস্টার করুন। সেখানে আপনি একটি `firebaseConfig` অবজেক্ট পাবেন।
৪. **কী (Key) বসানো**: আপনার ফোল্ডারের `js/firebase-config.js` ফাইলটি ওপেন করুন এবং সেখানে আপনার সংগৃহীত `apiKey`, `projectId` ইত্যাদি তথ্যগুলো বসিয়ে সেভ করুন।
৫. **ডাটাবেস চালু**: মেনু থেকে 'Firestore Database' এ যান এবং 'Create Database' এ ক্লিক করুন। এরপর 'Build in test mode' সিলেক্ট করুন এবং একটি 'Collection' তৈরি করুন যার নাম দিন `admissions`।

**ব্যাস!** এখন কোনো ছাত্র ভর্তি ফরম পূরণ করলে তা সরাসরি আপনার অ্যাডমিন প্যানেলে চলে আসবে এবং আপনি ছাড়া আর কেউ তা দেখতে পারবে না।
